<br><br>
+48 is the area code for Poland<br>
Please change it to the code for your country.
<br><br>
<b>The code is for the Arduino IDE</b>
<br>

Connect TTP223 to TTGO T-Call 1.3<br>
VCC to 5V<br>
GND to GND<br>
OUT to 22 PIN
